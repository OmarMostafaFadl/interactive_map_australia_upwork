Here are the steps to use this tool
===================================

STEP 1
======

Install need Libraries as follow: 

1. pandas
2. numpy
3. django
4. plotly.express


--------------------------------------------------------
STEP 2
======
Clone this repo: https://github.com/OmarMostafaFadl/interactive_map_australia_upwork.git


--------------------------------------------------------

STEP 3
======
1. Open maptool
2. Open a terminal in this directory
3. Type this command: python manage.py runserver
4. Open your browser and enter for the landmarks tool: http://127.0.0.1:8000/
5. Open your browser and enter for the cafes tool: http://127.0.0.1:8000/cafes


STEP 4
======
Choose from the filters and click submit.
Wait for the map to open in a new tab.
